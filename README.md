# Arc Network Monitor Tool 🟢⚙️

A professional-grade monitoring utility built for the **Arc Layer-1 Stablecoin Hackathon**. This tool provides real-time visibility into the Arc network's gas consumption and transaction throughput.

## Built for Arc
- **Real-time Gas Tracking**: Monitor L1 gas fluctuations to optimize transaction timing.
- **Block Metrics**: Log transaction density and block intervals for performance analysis.
- **Hackathon Optimized**: Designed specifically for developers participating in the Arc/Circle ecosystem.
